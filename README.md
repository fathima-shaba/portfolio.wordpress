# Portfolio

Welcome to my personal portfolio website! This project showcases my skills, experience, and projects as a BSc Computer Science student and aspiring Full Stack Developer.

## Live Demo
You can view the live website here: [https://fathima-shaba.github.io/portfolio.wordpress/]

##  Tech Stack
* **HTML5**: Used for the structure of the website.
* **CSS3**: Used for styling, including custom animations, glowing effects, and responsive design.
* **GitHub Pages**: Used for hosting the website live.

##  Key Features
* **Responsive Design**: Optimized for both desktop and mobile devices.
* **Modern UI**: Featuring custom glassmorphism effects, animations, and a dark theme.
* **Resume Download**: Easily accessible link to download my professional resume.
* **Project Showcase**: A dedicated section to highlight my development projects.
